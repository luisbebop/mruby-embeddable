mruby-embeddable
=========

## Description

Run mruby in micro-controllers in an easy way. Support: Arduino Due and others (soon!) 

## Features

* mirb connected to the microcontrollers to rapid development.

## Requires

* Arduino 1.5.4

## Install

```ruby
ruby make_mruby.rb
ruby make_arduino.rb
```

## Usage
```ruby
ruby make_arduino.rb connect
```
